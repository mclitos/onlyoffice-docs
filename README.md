# onlyoffice-docs
OnlyOffice Document Server - ONLYOFFICE - Docker


### Clona el Repositorio
```
git clone https://github.com/mclitos/onlyoffice-docs
```

### entra en la carpeta 
```
cd onlyoffice-docs
```

### Levanta el Cntenedor 
```
 docker compose up -d
```

> [!IMPORTANT]
> Modificar línea 19. "- JWT_SECRET=my_secret" cambiar my_secret por un token puede ser de 32 bits 


Dirección de ONLYOFFICE-Docs: "http://localhost:8000" ó "http://IP:8000" ó "https://tu.dominio.com"

Si ves el sitio web, es que funciona bien. No hay que hacer nada más

Sirve para realizar la conexión con: FileBrowser Quantum, NextCloud, OwnCloud, Seafile, etc.
